## 基础知识篇

目前主流的Windows笔记本搭载的都是X86/AMD64（AMD64又称X64）架构处理器，MacBook搭载的是苹果基于ARM架构魔改的M/A系列处理器。Windows阵营有ARM架构处理器，主要是高通骁龙X系列芯片，但是目前Windows On Arm（简称WOA）生态仍然不完善，所以本系列不列举WOA笔记本产品及其所搭载的芯片。当然，有些需求用WOA笔记本能够满足的，会在本系列特别篇中列出，希望能对你有所帮助。

无论是X86/64处理器还是Arm处理器，主要由CPU、GPU、NPU和I/O等部分组成，我们在此一一展开介绍。

### CPU

CPU，即Centre Produce Unit（中央处理单元），是电脑的最核心部件，用于处理绝大多数指令、调配资源甚至渲染部分2D画面。CPU的优劣，主要看核心数、频率、能效和缓存这几个方面。

#### 核心数

目前的处理器核心主要由P核（即性能核）和E核（即能效核）组成（偶尔还有LPE，即超低功耗能效核）。P核性能更强，可以处理更高负载的任务；E核性能较弱，用来处理低负载任务。以Intel Core Ultra 5-225H处理器为例，这款处理器为4P 8E 2LPE核心，一共14个核心。其中决定性能的是4P部分，有高性能需求的同学可以简单理解为4核处理器。

#### 频率

频率，决定了一个核心在一定时间内能处理的指令数。简单理解，一般情况下，处理器频率越高，性能越强。以Intel Core Ultra5-225H和Ultra 7-255H为例，225H的P核睿频为4.9GHz，255H的P核睿频为5.1GHz，255H的P核比225H的P核性能更强。

#### 能效

能效，决定了处理器的耗电情况。在性能相近的情况下，能效越高，处理器功耗越低，对笔记本散热的压力也越小。但是目前市面上少有测试笔记本处理器能效的测试，一般都是测试满载性能，所以在此不过多赘述。

### GPU

目前集成在处理器中的GPU普遍性能较弱，只有特殊型号比如AMD Ryzen AI MAX+ 395、Intel Core Ultra X7-358H这类处理器拥有高性能核显，用来满足部分用户对高性能核显的需求。一般情况下，不建议使用核显玩大型游戏（比如黑神话悟空之类），如果有这类需求，建议购买游戏本而非核显本（除了搭载上述处理器的高性能本）。

### NPU

NPU目前在Windows笔记本上基本没有什么用处，Windows大多数情况下不会调用NPU。笔者没有用过Macbook，也不太了解MacOS对NPU的利用情况，在此不过多赘述。

### 缓存

CPU缓存是位于CPU芯片上的高速存储器，用于临时存储CPU可能需要快速访问的数据。它通过减少CPU与主存储器（RAM）之间的交互时间，大幅提升系统性能。现代CPU通常包含三级缓存：L1、L2和L3，每一级缓存在速度、容量和数据命中率之间找到平衡。缓存的预读功能通过预测即将需要的数据，将其提前加载到缓存中，从而进一步减少延迟。

缓存并非越大越好。缓存采用昂贵的静态RAM（SRAM），增加容量会导致晶体管数量和发热量增加，设计复杂性也随之提高。如果设计不合理，可能导致缓存延迟，反而影响性能。因此，缓存大小的选择需根据具体使用场景权衡。例如，视频编辑等顺序处理任务对缓存的依赖较小，而游戏等任务则能从更大的缓存中获益更多。



目前Windows X86/64处理器主要有两大阵营，一个是Intel阵营，一个是AMD阵营。
Intel阵营处理器的主要优势是单线程性能和高端处理器的能效，凭借更高的睿频和更大规模的P核，在游戏场景中提供更强劲的性能，其最新处理器PTL系列（即Ultra 300H系列处理器）的能效远优于AMD最新处理器；AMD阵营处理器的主要优势是多核性能以及低价位处理器的能效，在视频渲染等任务中AMD处理器的表现更好，搭载AMD处理器的低价位段笔记本续航一般也优于同价位段的Intel笔记本。

Intel官网介绍：

作为Intel Ultra系列处理器的开山之作，Ultra 100H系列的表现只能说惨不忍睹。取消超线程过后，无论是单核还是多核，Ultra 100H系列处理器的CineBench分数都不如同级的13代酷睿标压；异构处理器的续航问题在这代也未能得到解决，部分续航长的型号全靠OEM厂商刻意压低离电性能。哪怕Intel刻意强调NPU等AI方面的进步，也难以掩盖Ultra 100H拉跨的事实。尤其是在今年的行情下，新出的搭载Ultra 100H处理器的笔记本普遍比库存的Ultra 200H笔记本还要贵，因此，搭载这个系列处理器的笔记本笔者完全不推荐。

#### Ultra 300H系列（PTL处理器）\[6结尾]

Intel官网介绍：

作为PTL系列处理器的中端标压小核显版本，这几颗处理器本质是Intel Core Ultra 300X系列去掉大核显的减配版本。这意味着，这几颗处理器的CPU部分和Ultra 300X系列是一样的。其中，Ultra 5-336H是4P 4E 4LPE的核心配置，Ultra 7和Ultra 9都是4P 8E 4LEP的核心配置。这几款处理器在CPU性能强于Ultra 200H系列处理器的同时，综合续航强于Ultra 200V系列低压处理器，同时GPU性能足以应对日常办公和影音娱乐场景，价格合适的话不失为值得考虑的选择。

当然，部分轻薄取向的游戏本OEM选择了这款处理器，主要是为了长续航，如果你在考虑购入一款续航和性能都很强劲的游戏本，可以考虑搭载这几款处理器的型号。
