# CLAUDE.md

This file provides guidance to Claude Code (claude.ai/code) when working with code in this repository.

## 项目说明

这是北邮数码社撰写的笔记本推荐系列文章，面向高考新生选购笔记本。所有内容为 Markdown 文档，非代码工程。

## 文件结构

- `0.笔记本推荐榜单.md` — 总榜单，包含所有推荐机型的优劣势摘要
- `模板.md` — 单篇机型的写作模板
- `声明.md` — 免责声明

三个品类文件夹，每个下有按价位段划分的子文件夹：

```
轻薄本/游戏本/全能本/
├── 1.XXXX元以下/
├── 2.XXXX-XXXX元/
├── ...
```

- **轻薄本**：≤1.5KG，无独显。超轻本（≤1.1KG）在榜单中用粗体标注
- **游戏本**：搭载独显且支持独显直连。轻薄游戏本（≤2.0KG）在榜单中用粗体标注
- **全能本**：介于轻薄本和游戏本之间。高性能全能本（TDP>55W）用粗体标注

## 文件命名规范

`【品牌】型号名称.md`

品牌使用中文名（Apple 除外），品牌排序按拼音首字母：Apple → 惠普 → 华硕 → 机械革命 → 雷神 → 联想 → 荣耀 → 小米（红米归入小米）
